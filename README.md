# Management_System


## Introduction
这是一个基于SSM的权限管理系统，用户可以管理、分配权限。。。。

## 运行环境和所需工具
1. 编译器：IntelliJ IDEA
2. 项目构建工具：Maven
3. 数据库：Mysql
4. JDK版本：jdk1.8
5. Tomcat版本：Tomcat9.x

## 技术栈
1. IOC容器：Spring
2. Web框架：SpringMVC
3. ORM框架：Mybatis
4. 数据库：MySql
5. 安全框架：
6. 分页插件：
7. 数据源：Druid
8. 日志：log4j
9. 前端框架：Layui

## 功能
1. 管理增删改查
2. 模糊查询
3. 权限分配
4. 头像上传
